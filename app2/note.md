我今天想要做一個房價app, 用戶點進來後可以馬上根據公開免費的房價api 顯示附近的平均房價, 根據屋齡和一坪的價格來顯示不同顏色, 重點是app 的顯示畫面要是可愛, 3D, 簡潔風格.

App 需要每月匯入資料（CSV → DB）、空間查詢（附近平均房價）、聚合計算（各區每坪均價、依屋齡/價格上色），以及快速讀取給用戶使用。

用postgres搭配 PostGIS 可輕鬆處理「附近」查詢、計算平均價、依屋齡上色等。未來用戶變多也能輕鬆擴展。

後端：FastAPI + PostgreSQL。
前端：React Native + Three.js 做可愛 3D 房子模型 + 簡潔 UI。
地圖：Leaflet 或 Mapbox 加上顏色熱力圖。
